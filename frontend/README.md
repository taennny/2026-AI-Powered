# Roame

걷기만 해도 내 하루가 기록된다.

위치 기반 이벤트 탐색 앱 Roame의 프론트엔드 레포지토리입니다.

## ✨ 주요 기능

- 이벤트 조회
- 이벤트 생성
- 위치 기반 서비스

## 🛠 기술 스택

- React Native (Expo)
- TypeScript
- Zustand
- Axios

## 📁 폴더 구조

src/
├─ app/
├─ services/
├─ components/
├─ hooks/
└─ utils/

## 🚀 실행 방법

```bash
npm install
npm run start
```
